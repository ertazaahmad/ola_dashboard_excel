# Excel Dashboard Project – Ride Booking Analysis

## 📊 Project Overview

This project is an **interactive Excel dashboard** built to analyze ride-booking data.
The goal of this project was to practice **data cleaning, data transformation, and visualization in Excel** while extracting meaningful insights from a messy dataset.

The dashboard provides a quick overview of ride performance, booking trends, revenue insights, and ride status distribution.

---

## 🎯 Objectives

* Analyze overall ride bookings
* Track total revenue generated
* Understand ride status distribution (Completed, Cancelled, etc.)
* Identify booking patterns and trends
* Build an interactive dashboard using Excel tools

---

## 🗂 Dataset

The dataset contains **ride booking records** with fields such as:

* Ride ID
* Date
* Vehicle Type
* Ride Status
* Pickup Location
* Fare Amount
* Payment Method
* Distance

Total records: **~5000 rows**

The dataset was intentionally messy to practice **data cleaning and transformation**.

---

## 🛠 Tools & Features Used

### Excel Features

* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Data Cleaning
* Sorting & Filtering
* Calculated Fields

### Data Preparation

* Removed blank rows
* Standardized ride status values
* Cleaned inconsistent entries
* Structured the dataset for analysis

---

## 📈 Dashboard KPIs

The dashboard highlights important metrics such as:

* **Total Bookings**
* **Total Revenue**
* **Completed Rides**
* **Cancelled Rides**
* **Revenue by Vehicle Type**

---

## 🔍 Key Insights

Some insights discovered from the dataset:

* Certain vehicle types generate higher revenue.
* Completed rides contribute the majority of revenue.
* Cancellation patterns can affect daily booking performance.
* Ride demand varies across different days.
